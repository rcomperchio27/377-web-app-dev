Sudoku

Fill in the missing numbers, each column can only have one of each number 1-9, each row can also only have one of each number 1-9, and each sub square can only have one of each number 1-9.

Click on the squares to change the number, when your done with the puzzle click the "check puzzle" button below to see if you are correct.

The Squares that are green have the correct number and the squares that are red have the incorrect number
